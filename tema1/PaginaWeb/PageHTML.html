Functionality 1: Structure and Hierarchy (H1 - H6)

HTML headings (from H1 to H6) define the content structure. H1 is used for the main title of the page, and H2, H3, etc., are used for subtitles and sections. H1 has the largest size by default; as the numbers increase, the size decreases.

Functionality 3: Images

The <img> tag is responsible for placing visual content. Unlike others, it is an empty tag (it does not require a closing tag), and it requires two essential attributes: "src", which indicates the path to the image file, and "alt", which describes the image for accessibility and when the image fails to load.

Lists allow content to be organized in a readable manner; here I will show 2 types: <ul> creates an unordered list (with bullets). <ol> creates an ordered list (with numbers or letters). Both use the <li> tag for each individual item within the list.

Functionality 5: Data Tables (<table>)

The <table> tag is used to structure data into rows and columns. To build a table, nested tags are required: <tr> defines a row within the table, and <td> defines a data cell within that row. Optionally, <th> is used for column headers.
