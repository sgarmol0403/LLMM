Funcionalidad 5: Tablas de Datos

La etiqueta <table> se usa para estructurar datos en filas y columnas. Para construir una tabla se requieren etiquetas anidadas: <tr> define una fila dentro de la tabla, y <td> define una celda de datos dentro de esa fila. Opcionalmente, se usa <th> para encabezados de columna.
