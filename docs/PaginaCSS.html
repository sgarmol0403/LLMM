Funcionalidad CSS 2: Fondo de Elemento (<background>)

La propiedad "background" es una abreviatura para varias propiedades de fondo (color, imagen, repetición, posición, etc.). Las sub-propiedades más comunes son "background-color" para establecer un color sólido detrás del elemento y "background-image" para usar una imagen como fondo.

Funcionalidad CSS 3: Bordes de Elemento (<border>)

La propiedad "border" se utiliza para agregar una línea alrededor de un elemento HTML. Es una propiedad abreviada que requiere tres valores clave: el "grosor" del borde (ej: 2px), el "estilo" (ej: solid, dotted, dashed) y el "color" (ej: black, #333). Se puede usar individualmente (border-top, border-left, etc.) para lados específicos.

Funcionalidad CSS 4: Márgenes Externos (<margin>)

La propiedad "margin" crea espacio en la parte exterior de un elemento (separándolo de otros elementos cercanos). Es la capa más externa del Modelo de Caja de CSS . Se puede usar la abreviatura (margin: top right bottom left) o propiedades específicas como "margin-top" o "margin-left". A diferencia del `padding`, el margen es siempre transparente y no afecta el tamaño real del contenido.

Funcionalidad CSS 5: Tipografía y Texto (<font-*>)

Las propiedades "font-" controlan el estilo y la apariencia del texto. "font-family" define el tipo de letra que se utilizará (ej: Arial, serif). "font-size" controla el tamaño del texto (ej: 16px, 1.2rem). Otras propiedades importantes incluyen "font-weight" para el grosor (negrita) y "font-style" para cursiva.
