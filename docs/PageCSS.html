CSS Functionality 1: Text Color (<color>)

The "color" property is used to define the color of the text within an HTML element; colors can be specified using names (e.g.: 'red') or hexadecimal codes (e.g.: '#FF0000').

CSS Functionality 2: Element Background (<background>)

The "background" property is a shorthand for various background properties (color, image, repeat, position, etc.). The most common sub-properties are "background-color" to set a solid color behind the element and "background-image" to use an image as a background.

CSS Functionality 3: Element Borders (<border>)

The "border" property is used to add a line around an HTML element. It is a shorthand property that requires three key values: the border **thickness** (e.g.: 2px), the **style** (e.g.: solid, dotted, dashed), and the **color** (e.g.: black, #333). It can also be used individually (border-top, border-left, etc.) for specific sides.

CSS Functionality 4: External Margins (<margin>)

The "margin" property creates space on the outside of an element (separating it from other nearby elements). It is the outermost layer of the CSS Box Model . The shorthand can be used (margin: top right bottom left) or specific properties like "margin-top" or "margin-left". Unlike `padding`, the margin is always transparent and does not affect the actual size of the content.

CSS Functionality 5: Typography and Text (<font-*>)

The "font-" properties control the style and appearance of text. "font-family" defines the typeface that will be used (e.g.: Arial, serif). "font-size" controls the text size (e.g.: 16px, 1.2rem). Other important properties include "font-weight" for thickness (bold) and "font-style" for italics.
